我叫張開永，目前碩二，預計6月畢。大學從臺科機械係畢業後便進入臺積電做設備工程師。 工作期間決定換跑道到資訊產業於是進入臺科醫學工程所的醫療影像分析實驗室。 研究主要是用python將醫療影響透過深度學習模型進行分析，也會對醫療影像做一些前處理。 就讀研究所期間也曾經參加數位能力培訓課程，學習後端java，java web，HTML UI 與資料庫規劃。 雖然目前已有一些程式設計的基礎但缺乏實作專案與軟體開發系統性的知識，因此希望能有機會參與WeHelp。
為了成為軟體工程師，曾做過什麼努力？
曾經在網路上自學python，碩一時曾參加數位能力培訓課程學習後端java， java web，HTML UI 與資料庫規劃。
如果參與這個計畫，會怎麼安排學習時間？
將會全職投入參加課程，努力更上課程進度。還有時間的話會自己找project做。
是否有想要加入的軟體公司？為什麼想加入該公司？
希望能加入趨勢科技，之前學校就業博覽會時有諮詢裡面的工程師， 他提到目前公司正在招募能夠運用AI及軟體的職缺，因為我目前研究領域跟AI相關， 如果能再更精進軟體的能力應該很有機會能夠加入。另外趨勢科技是一家組織規模較大的純軟公司， 若有幸能進入，除了能夠精進自己的實力外，也將對我的職涯發展有很大的助力。
請描述一件讓你產生明顯負面情緒的事情，你如何處理該情緒？
當事情不如預期或出現無法掌控的局面時，我會有負面情緒。 當事情發生時，我會提醒自己要轉念，讓自己專注在過程中所獲得的經驗，而非執著於結果。
關於這份申請網頁，請分享一個你開發時的技術心得。
透過製作申請網頁的過程，促使我思考開發出一個好的網頁及介面所需運用的技術， 例如我的網頁最後的排版是透過flexbox技術來切板的，主要是為了不爆板及讓文字區塊完美置中。 此網頁同時也使用了Response Web Design (RWD)的技術，能夠使網頁在電腦桌面、平板和 手機上呈現不同大小的文字和文字區塊的調整，使讀者能有更佳的閱讀體驗。
